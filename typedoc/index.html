  • for each child of element
    • start loop with index as starting point
    • if child's height accumulated with currentPosition is not larger than pdfPageHeight
      • add child's height to currentPosition
    • else if child's height accumulated with currentPosition is larger than pdfPageHeight
      • group all currently iterated children not including the current child
      • record the top position of the first grouped child
